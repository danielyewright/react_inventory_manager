## Installation Instructions

1. Clone this repo. `git clone https://github.com/mtallentb/FINAL_INVENTORY.git`

2. `cd FINAL_INVENTORY`

3. `npm install`

4. `npm start`

5. If the app doesn't pop up automatically, head to `http://localhost:3000`

## Prerequisites

#### Node.js

`brew install node`
